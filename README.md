# Netflix Data Immersion & Wrangling

## Project Overview
This project focuses on cleaning and preprocessing the Netflix Titles dataset obtained from Kaggle.

## Tools Used
- Python
- Pandas
- VS Code

## Data Cleaning Steps
1. Loaded the dataset.
2. Examined dataset structure.
3. Identified missing values.
4. Removed duplicate records.
5. Filled missing values in text columns.
6. Converted date_added column to datetime format.
7. Saved the cleaned dataset.

## Files Included
- netflix_titles.csv
- cleaned_dataset.csv
- data_cleaning.py
- Data_Dictionary.xlsx

## Outcome
A cleaned and structured dataset ready for Exploratory Data Analysis (EDA).
